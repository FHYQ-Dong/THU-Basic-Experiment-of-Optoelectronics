# 实验报告撰写方案

## 一、Exp3 写作风格总结

通过分析已完成的 Exp3 报告，总结写作风格如下：

| 维度     | 风格                                                                                         |
| -------- | -------------------------------------------------------------------------------------------- |
| 文档类   | `ctexart`, `twocolumn`, A4, scale=0.8                                                    |
| 页眉     | 左：实验名称简称；右：无37~董皓彧~2023010659                                                |
| 页脚     | 居中页码 `---~~\thepage~~---`                                                              |
| 标题     | `\title{XXX实验报告}` + `\author{无37~董皓彧~2023010659}` + `\date{\zhtoday}`          |
| 正文结构 | **不写实验原理**，直接从"实验过程及数据分析"开始，每个实验任务对应一个 `\subsection` |
| 数据展示 | 表格用 `booktabs`（`\toprule/\midrule/\bottomrule`），浮动体 `[H]`                     |
| 图片     | `\includegraphics[width=\linewidth]`，紧跟 `\vspace{-2em}` 压缩间距                      |
| 文字描述 | 简洁、以数据和观察为主，定量结果用行内公式给出（如阈值电流、斜率效率）                       |
| 数据处理 | Python 脚本（numpy + matplotlib）放在 `scripts/` 下，生成图片放 `images/`                |

### 示波器相移分析

示波器 Ch1（黄色）为驱动正弦信号（13.9V峰峰值），Ch2（绿色）为光电探测器输出。
根据理论公式 (1-18)(1-19)：
- φ_D = π/2 时：I ∝ 1 + 2J₁(φₘ)sinωt，输出与输入**同频同相**
- φ_D = -π/2 时：I ∝ 1 - 2J₁(φₘ)sinωt，输出与输入**同频反相**（相移180°）

| 图片 | 相移 | 输出Vpp | 对应偏置 | 说明 |
|---|---|---|---|---|
| `6.6.jpg` | 6.6° ≈ 0° | 2.53V | φ_D = π/2 | 同频同相，信号幅度最大 |
| `179.0.jpg` | 179.0° ≈ 180° | 1.54V | φ_D = -π/2 | 同频反相 |
| `-177.1.jpg` | -177.1° ≈ 180° | 970mV | φ_D = -π/2 | 同频反相（另一次测量） |

179.0° 和 -177.1° 本质均为反相（≈180°），可能是旋转 1/4 波片到不同位置的两次测量。

### 撰写方案

```
\section{实验过程及数据分析}
```

```
\subsection{点到点光纤通信系统}
  - 列表/表格给出两个波长的发射功率和接收灵敏度
  - 计算系统余量 = 发射功率 - 接收灵敏度
  - 结合光纤衰减系数计算最大传输距离
    - 1310nm: 余量 = -6.20 - (-36.50) = 30.30 dB, 距离 = 30.30/0.35 ≈ 86.6 km
    - 1550nm: 余量 = -2.92 - (-34.45) = 31.53 dB, 距离 = 31.53/0.2 ≈ 157.7 km

\subsection{视频光纤传输}
  - 插入 2-video-trans.jpg，说明视频信号可正常传输
  - （视频系统灵敏度与数字系统相同或类似，可合并讨论）

\subsection{双波长 WDM 系统}
  - 表格给出两个 WDM 器件的插损测量数据（从 3-4-插损.png 转录）
  - 计算包含 WDM 插损后的系统余量和最大传输距离
  - 分别讨论同向传输和双向传输

\subsection{WDM 隔离度}
  - 从插损数据中提取信道串扰数据
```
