# QKD-BB84协议实现-FPGA控制

## 背景

实验需要实现基于空间光学的BB84协议。具有4个激光器、4个SPD。Alice和Bob的发射基、接受基选择已经硬编码在光路中（即每个激光器/SPD）与一个基（0、90、+45、-45）绑定。

实验有一个TCSPC，将4个SPD电信号接入TCSPC即可得到每个单光子事件的时间戳。TCSPC需要同步信号输入。因此Alice和Bob间需要同步时钟。

## FPGA功能

- 产生同步信号
- 通过UART从Alice计算机处收取随机产生的bit串作为即将发送的0/1和随机选择的偏振基。
- 根据UART收取的随机bit串控制4个端口的电平（连接到4个激光器）

## 要求

- FPGA型号：xc7a35tfgg484-2，系统时钟100MHz
- 使用 Verilog 实现
- 同步信号产生模块产生的同步信号频率可调
- 通过UART收取的数据应当临时存储在FPGA内部
- 激光器控制模块控制激光器发光的频率应当基于同步信号，频率可调。
